序号········类型与描述  <br/>
1···········基本类型：它们是算术类型，包括两种类型：整数类型和浮点类型。<br/>
2···········枚举类型：它们也是算术类型，被用来定义在程序中只能赋予其一定的离散整数值的变量。   <br/>
3···········void 类型：类型说明符 void 表明没有可用的值。   <br/>
4···········派生类型：它们包括：指针类型、数组类型、结构类型、共用体类型和函数类型。   <br/>
