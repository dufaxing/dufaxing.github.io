### 列表和字符串的互相转换

> 利用切片操作，实现一个trim函数，去除字符串首尾的空格。

```
#!/usr/bin/python3
def trim(s):
    temp = list(s)#字符串转列表
    for i in temp:
        if (temp[0] == ' '):
            temp = temp[1:]
        if temp[0] != ' ':
            break
    for j in temp:
        if (temp[-1] == ' '):
            temp = temp[:-1]
        if temp[-1] != ' ':
            break
    temp = ''.join(temp)#列表转字符串
    return temp
print(trim('    dufaxing  '))
```
output:<br/>

```
dufaxing
```
